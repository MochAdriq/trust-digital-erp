An internal Enterprise Resource Planning (ERP) system designed to streamline the distribution and inventory tracking of digital assets for Trust Digital. This system replaced manual spreadsheets with a robust, automated dashboard.

## ‚ö° Key Features
* **Automated Distribution:** Algorithm to automatically assign unique account profiles (A-E) to customers based on stock availability.
* **Role-Based Access Control (RBAC):** Secure login and dashboard separation for Admins vs. Operators using JWT.
* **Real-time Inventory:** Dashboard analytics to monitor stock levels (Private, Sharing, VIP) in real-time.
* **Data Resilience:** Custom-built offline Backup & Restore utility using JSON processing.
